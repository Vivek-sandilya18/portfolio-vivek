This portfolio is designed to be a clean, responsive, and professional representation of my work. It highlights my technical proficiency in the Java ecosystem, frontend development, and my ongoing growth in software engineering.

## 🛠 Tech Stack
* **Structure:** HTML5
* **Styling:** CSS3 (Custom responsive layout)
* **Fonts:** Inter (via Google Fonts)

* **Responsive Design:** Fully mobile-friendly layout.
* **Interactive UI:** Smooth navigation and clear sections for Experience, Skills, and Projects.
* **Project Showcase:** Highlights of my work including Student/Library Management Systems and Blogging applications.
* **Social Connectivity:** Direct links to GitHub, LinkedIn, and Email.
* **Timeline View:** A visual representation of my learning journey from 2023 to 2026.
